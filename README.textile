h1. Fork of Play Framework 1.5 with added ability to include files in the build war

Get additional info by typing the command: **play war help**

Refer to this section:

~ --include:
~ Includes given source_[file/directory]#destination_directory pairs into the WAR output path. Source can be a file or a directory while destination must be a directory!
~ Example: --include "C:\Users\stano\Documents\OneDrive\Work\Fastrope\mhg-salesforce-sync-java\.ebextensions#.ebextensions pom.xml#WEB-INF/application/pom"

**For installation instructions refer to: https://github.com/playframework/play1**
